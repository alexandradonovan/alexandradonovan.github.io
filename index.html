Rat Sightings in NYC

Rat Sightings

Summary

  • For our data journalism project, we wanted to display which zip codes in Manhttan reported the most rat sightings per their respective areas. It's important to note that just because an area reports more rats, it does not mean that that area actually has more rats, only that people are seeing them and calling in complaints to the Department of Health and Mental Hygiene. The colors on our macro map correspond to which areas reported how many rat sightings per the zip code areas. The more pigmented (darker red) a zip code is, the more rat sightings were reported on average. Originally we had intended to include an option where the user hovers over the zip code to see information for that zip code but instead the micro summaries for 10 zip codes can now be found on our micro summaries drop down page. The micro summaries visually represent the rat sightings for a given zip code.
  • We found that the zip code with the highest density of rat sightings per square mile was the zip code 10026 followed by neighboring zip code 10025.
  • The density of rat sightings reported in Central Park (Zip: 00083) was very low suggesting that people don't generally feel the need to report rat sightings when they are in nature.
  • Zip code 10048 had no rat sightings which makes sense because it is very small.

Potential flaws in data collection and interpretation:

  • Not everyone is aware that when they see a rat, they can report a sighting to the Department of Health and Mental Hygiene meaning that not every rat that is sighted is reported.
  • Zip code boundaries are very specific and therefore difficult to accurately map, additionally they change with some regularity.
  • The exact square foot measurement of zip code areas also shifts and that information is not always readily available.
